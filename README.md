# 🏥 Explainable AI for Healthcare Diagnosis

## 📌 Project Overview

This project develops an Explainable AI (XAI) based healthcare diagnosis system using Machine Learning.

Unlike traditional ML models that only provide predictions, this application explains **why** a prediction was made using SHAP and LIME, making the AI more transparent and trustworthy.

---

## 🎯 Objectives

- Predict diseases using Machine Learning.
- Explain predictions using SHAP.
- Explain individual predictions using LIME.
- Improve transparency and trust in AI-assisted healthcare.

---

## 🛠 Technologies Used
